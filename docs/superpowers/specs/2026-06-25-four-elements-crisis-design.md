# 《四象天灾阵》四集连续故事设计方案

本项目是针对睡前故事《四象天灾阵》（第23集 - 第26集）的4集连续故事策划案。该系列将以B组（六小只）为主导，打屁巫师为核心辅助，双公主在后方支持，讲述一场席卷森林的特大危机，并最终以打屁巫师的魔法本源为牺牲解决灾难。

---

## 一、 基本设定

### 1. 角色与阵营分工
*   **主角团主导（B组）**：小粉 (Xiaofen)、小绿 (Xiaolu)、小圆 (Xiaoyuan)、卡皮 (Kapi)、小熊熊错误 (Xiaoxiong)、球球 (Qiuqiu)。
*   **主要辅助**：打屁巫师 (Dapi Wizard)。由于在第22集失去了紫色斗篷，他此时身穿滑稽的绿白相间条纹背心，裹着碎花床单。
*   **后方支援**：善良公主、海蓝公主。在树屋内进行远程监测，并保管口袋物品，在第26集决战关键时刻提供道具投送。

### 2. 危机来源：天才威的“四象天灾阵”
天才威启动了四台相互关联、可发生连锁过载的巨型环境控制仪，试图在森林建立“绝对实验场”。四台机器分别为：
*   **风**：巨型飓风涡轮仪（森林-044号 / 第45台机器）
*   **雷**：脉冲雷雨机（森林-045号 / 第46台机器）
*   **地/重力**：重力地裂仪（森林-046号 / 第47台机器）
*   **熔岩聚变/核心**：熔融能量舱（森林-047号 / 第48台机器）

## 二、 分集大纲与悬念设计

### 1. 第23集：狂风中的坠落 (Fall in the Gale - 风)
*   **目标机器**：森林-044号机器（巨型飓风涡轮仪）。
*   **故事地点**：森林西南偏西。
*   **核心情节**：强风肆虐，森林树木被成片吹倒，树屋无法低空悬停。B组在地面逆风前行，利用卡皮的高密度身体和球球的飞行避开倒塌的红杉树。
*   **关闭方案**：使用口袋中的 `半恐龙半吸血鬼牙齿` (强行卡住高速旋转的涡轮风叶齿轮) + `彩虹石子` (释放内部的松树和小马烟雾，通过烟雾轨迹将风力导向反方向，使机器自毁)。
*   **留扣子（悬念）**：机器停转时释放出强烈的反向气旋，球球为了推开险些被卷走的小粉，自己被暴风卷入天空中的滚滚雷云里。此时指南针亮起明红光芒，指向正南方向——雷电机器启动，且球球的生命定位信号恰好在雷暴中心！
*   **纪念品**：`狂风石笛` (Wind Stone Whistle)
    *   *口袋互动*：风吹过 `珍珠母薄片` 发出风铃般的和声；吹动 `粉色星星糖` 在口袋里沙沙滚动，发出安神的微响。

### 2. 第24集：怒雷下的牢笼 (Cage Under the Thunder - 雷)
*   **目标机器**：森林-045号机器（脉冲雷雨机）。
*   **故事地点**：森林正南方向。
*   **核心情节**：天空下起大范围的金属酸雨，强雷电在低空蔓延。六小只在打屁巫师的掩护下，找到了被困在强电磁铁笼里的球球。洞洞幺驾驶着雷电发射器在空中疯狂防守。
*   **关闭方案**：使用 `静电结晶电池` (灰色石头，用于吸收电磁场的超额电荷，吸收后重新充能为蓝色活性状态) + `银铃铛碎片` (产生共鸣，将劈向机器的雷电折射，击毁机器的能量接收板)。
*   **留扣子（悬念）**：雷雨机被短路自毁，但洞洞幺在最后关头按下了天罗地网防卫按钮，降下高压雷网将小圆困住并俘虏。小圆在被抓前拼尽全力将“四次元口袋”抛给了小粉。紧接着，地面发生强烈的震动，指南针指向西南——重力地裂机器激活。
*   **纪念品**：`避雷金磁石` (Lightning-repelling Magnet)
    *   *口袋互动*：吸取蓝色 `静电结晶电池` 上的微小电火花，通过 `发光白菜棋子` 折射成一个半透明的蓝色静电防护罩，在口袋里保护作物和种子。

### 3. 第25集：崩裂的重力谷 (The Cracking Gravity Valley - 地/重力)
*   **目标机器**：森林-046号机器（重力地裂仪）。
*   **故事地点**：森林西南方向。
*   **核心情节**：重力法则在森林局部崩塌，地面撕裂成无数深不见底的峡谷，重力时而变为原来的十倍（把人压得动弹不得），时而变为零（全员飘在空中）。他们必须携手在变幻的重力场中救出小圆并关闭地裂仪。
*   **关闭方案**：使用 `橡果` (卡在引力泄压阀阀口防止其闭合) + `发光白菜棋子` (彩虹光拉伸形成重力稳定的“光桥”，让大家在峡谷崩塌前撤离)。
*   **留扣子（悬念）**：在撤离峡谷的最后一刻，地缝开始大面积崩塌，卡皮与小圆所在的铁笼正向深渊滑落。小绿为了在终端强行插入密钥并固定泄压阀，主动留在随时会塌陷的对岸。小绿成功输入密钥稳定了光桥，但自己被天才威的机械兵团包围俘虏。此时，森林正中心的岩浆翻涌，一座钢铁熔岩要塞——熔融能量舱升起，核心聚变进入10分钟自毁倒计时。
*   **纪念品**：`重力浮石` (Gravity Pumice)
    *   *口袋互动*：`七彩剪纸蝴蝶` 停留在浮石上在口袋空中飘浮，`彩虹石子` 释放的微量烟雾围绕其旋转，形成微型小行星带。

### 4. 第26集：熔岩中的新生 (Rebirth in the Molten Lava - 核心聚变)
*   **目标机器**：森林-047号机器（熔融能量舱）。
*   **故事地点**：森林正中心。
*   **核心情节**：仅剩的小粉、卡皮、球球以及打屁巫师，必须潜入熔岩能量舱。球球、小圆、小绿被吊在熔岩池上方的铁笼中，舱内温度正急剧上升。
*   **关闭方案**：双公主在树屋通过传送光束投送口袋中的 `冰川极光珠` (释放绝对零度寒能冻结反应堆) + `七彩画意泡泡` (将聚变爆炸的余温与冲击波强行收纳进泡泡的二维画幅中)。
*   **障碍与牺牲**：铁笼的电磁锁只有巫师魔法才能解除。打屁巫师为了救出三个伙伴，强行透支了自己仅存的所有魔法本源，以至于他的身体和花床单都冒出了黑烟。救出同伴后，打屁巫师彻底失去了巫师魔法，变为了一个完全没有魔力的“凡人”，自此只能靠嘴皮子和“高级顾问”身份行动。
*   **终局画面**：危机解除，天才威和洞洞幺在废墟中狼狈传送逃跑。打屁巫师虽然没了魔法，但昂首挺胸表示：“没有魔法的顾问才是高级顾问！”
*   **纪念品**：`熔融结晶芯` (Molten Crystal Core)
    *   *口袋互动*：极光珠与画意泡泡已被消耗。熔融结晶芯与 `发光白菜棋子` 共鸣在口袋天花板投影出暖红色的烟花，与 `彩虹石子` 结合，使口袋世界犹如壁炉般温暖，陪伴种子与 `小熊熊错误`。

## 三、 写作质量保障与红线控制

在撰写这4集连续故事的过程中，必须严格遵守以下规则：
1.  **字数控制**：每集字数均需严格在 **2000字以上**。
2.  **口癖控制**：每集叙事中“不是X，是Y”口癖不得超过 **4次**（目标控制在0-1次），禁用“不是A也不是B而是C”。
3.  **对话平衡**：坚决避免连续多人对话以防TTS朗读混淆；任何描述段落不得连续超过 **2段** 纯旁白，必须穿插台词。
4.  **关闭方案合规**：关闭机器必须使用 **至少两样已有的口袋物品**，并且伴随牺牲与障碍（不得仅按按钮）。
5.  **定稿审核**：写完每集故事后，使用 `@deepseek-pro` 子代理进行逐项退化迹象检查，达标（评分>=4且退化迹象<=1）后方可提交并更新手册与历史记录。
